套件相關資訊

新增功能 🚀

  1. 提供較佳的語法來 透過命令複寫預設參數

  2. 提供較佳的語法，在組態檔聲明參數 ~

  3. 支援多樣的 配置轉換, 包括轉換至 omegaconf, dataclasses, dict, …, 等等，或轉換回 easy-configer!!

  4. 支援環境變數 (透過 os.Env 語法) 插值!!

  5. 動態 子組態 加載..

  6. 也於文件中新增一些 「快速上手」 範例 ~
