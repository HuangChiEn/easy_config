パッケージ関連情報

新機能 🚀

  1. v2.5.4は基本的には問題ありませんが、いくつかの既知の問題が残っていますので、v2.6を安定版としてリリースする予定です。

  2. ${cfg}と${env}でを引数として適用し、それぞれコンフィグのパラメータと環境の補完表記として使用します。

  3. AttributeDict　コンテナを適用します（これは純粋なPythonの辞書を継承しており）非フラットな引数を保存するために使用されます！

修正されたバグ 🐛

AttributeDict にホットフィックスを適用し、NoAttributeError を発生させるようにしました。 コンフィグのパラメータの上書きバグを修正し、allow_overwrite をサポートするフラグを追加しました。

簡易ユニットテスト 🧪

GitHubからこのプロジェクトをクローンした場合、以下のテストコマンドを実行してみてください。 python -m unittest discover

すべてのテストケースをtestフォルダーに配置しました。
