ダウンロードの手順

どのようにダウンロードするか？ ⚙️


  1. pypiにダウンロード

    コマンドプロンプトで簡単に　pip install easy_configer　コマンドを入力します（パッケージ名の衝突のため、pypiでは異なるパッケージ名が使用されます）

  2. ソースコードでビルド

    GitHubからプロジェクトをクローンしてローカルに取得します： git clone https://github.com/HuangChiEn/easy_config.git クローンしたプロジェクトのルートディレクトリに移動し　pip install -e .　を入力します。

  3. importの構文

    パッケージをインストールした方法に関係なく、次のように記述します： from easy_configer.Configer import Configer

パッケージの依存関係 🏗️

このパッケージは最初に Python 3.8 で作成され、3.6+ バージョンで動作することがテストされています。 当然のことながら、この軽量な解決策は　どのサードパーティパッケージにも依存していません。 私は Python 標準ライブラリのパッケージのみを使用しています（たとえば、pathlib、sys、...など）。 したがって、パッケージの依存関係について心配する必要はありません〜

ただし、IO_Converter機能を使用して構成オブジェクトを他のパッケージ（たとえば、omegaconfなど）に変換する場合は、omegaconfパッケージをダウンロードし、関連するバージョンに注意してください〜
