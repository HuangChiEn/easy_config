パッケージ関連情報
===================

新機能 🚀
^^^^^^^^^^^

#. 
   v2.5.4は安定バージョンですが、このバージョンではさらに多くのテストケースを追加しました（現在はより安定しています）~

#. 
   ${cfg}と${env}でを引数として適用し、それぞれコンフィグのパラメータと環境の補完表記として使用します。

#. 
   AttributeDict　コンテナを適用します（これは純粋なPythonの辞書を継承しており）非フラットな引数を保存するために使用されます！

修正されたバグ 🐛
^^^^^^^^^^^^^^^^^

AttributeDict にホットフィックスを適用し、NoAttributeError を発生させるようにしました。
コンフィグのパラメータの上書きバグを修正し、allow_overwrite をサポートするフラグを追加しました。

簡易ユニットテスト  🧪
^^^^^^^^^^^^^^^^^^^^^^

GitHubからこのプロジェクトをクローンした場合、以下のテストコマンドを実行してみてください。
``python -m unittest discover``

..

   すべてのテストケースをtestフォルダーに配置しました。
